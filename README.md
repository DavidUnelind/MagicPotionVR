# MagicPotion

## Overview

MagicPotion is a Unity-based VR potion-mixing game developed for the Meta Quest 3. Set in a lively fantasy tavern, the game challenges players to prepare and serve magical potions using immersive hand interactions and motion controls.

The game was created by **David Unelind, Moa Gassilewski, Klara Högberg, Johanna Fridh, and Linda Lohk-Blom** for the course **MAMF45 – Virtual Reality in Theory and Practice**.

## Gameplay

- Customers arrive and line up at the bar.
- Each guest requests a randomized potion order: **Love**, **Luck**, **Strength**, or **Health**.
- Players collect the correct ingredients and add them to the cauldron.
- Once the required ingredients have been added, the player must stir the cauldron with a stick to complete the potion.
- After the potion is fully prepared, the player presses the serve button to deliver the drink.
- Each successful serve awards points.
- The game runs as a timed session, with the objective of serving as many customers as possible before time runs out.

## Running the Project

1. Open `MagicPotionVR.slnx` or the Unity project folder in Unity.
2. Load the main scene from `Assets/Scenes/`.
3. Ensure XR and Oculus support packages are installed and configured.
4. Enter Play mode to test the VR potion-serving gameplay.

## Installing and Playing the APK on Meta Quest 3

1. Build the APK from Unity or use the provided APK file.
2. Open SideQuest on your PC.
3. Connect your Meta Quest 3 headset to the PC via USB.
4. Use SideQuest to transfer the APK to the headset.
5. Put on the headset.
6. Open the `Library` app.
7. Go to the menu and select `Unknown Sources`.
8. Find and launch `MagicPotion` from the list.
